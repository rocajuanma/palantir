# Palantir Demo

A simple demonstration of the Palantir terminal formatting package.

## Run the Demo

```bash
# From project root(recommended)
go run ./cmd/demo

# Or build and run
go build -o palantir-demo ./cmd/demo
./palantir-demo
```
